## 📌 Project Overview

In this project, I analyzed a pizza sales dataset using SQL to extract meaningful business insights related to sales performance, customer ordering behavior, and revenue trends. The analysis focuses on identifying top-performing pizzas, peak order timings, category-wise revenue contribution, and high-value customer orders.

This project also demonstrates the practical use of various SQL concepts such as aggregations, subqueries, Common Table Expressions (CTEs), window functions, ranking functions, and date-time functions.

## 📂 [Dataset Information](https://raw.githubusercontent.com/amnbl109/pizza-sales-sql-analysis/refs/heads/main/pizza_sales.csv)

The dataset contains transactional sales records of a pizza store, including:

* Order Details ID
* Order ID
* Pizza ID
* Quantity
* Order Date
* Order Time
* Unit Price
* Total Price
* Pizza Size
* Pizza Category
* Pizza Ingredients
* Pizza Name

## 📊 Business Problems & Analysis

### [1. Top 5 Highest Revenue-Generating Pizzas](02_top_5_pizzas_by_revenue)

Analyzed total revenue generated by each pizza to identify the top 5 best-performing pizzas based on sales revenue.

#### SQL Concepts Used

* `SUM()`
* `GROUP BY`
* `ORDER BY`
* `LIMIT`

### [2. Days and Hours with Highest Order Volume](03_top_ordering_days_and_hours)

Performed time-based analysis to identify the busiest weekdays and peak ordering hours based on customer order volume.

#### SQL Concepts Used

* `DAYNAME()`
* `HOUR()`
* `COUNT(DISTINCT)`
* `GROUP BY`
* `ORDER BY`

### [3. Revenue Contribution by Pizza Category and Size](04_revenue_by_pizza_category_and_size)

Analyzed how different pizza categories and pizza sizes contribute to overall revenue to better understand customer preferences and sales distribution.

#### SQL Concepts Used

* `SUM()`
* `ROUND()`
* Percentage Calculations
* `GROUP BY`
* `ORDER BY`

### [4. High-Value Order Analysis](05_high_value_orders)

Identified orders with spending higher than the average order value to detect premium customer purchases and high-value transactions.

#### SQL Concepts Used

* `SUM()`
* `AVG()`
* `GROUP BY`
* `HAVING`
* Subqueries
* `ORDER BY`

### [5. Category-Wise Pizza Revenue Ranking](06_pizza_revenue_ranking)

Ranked pizzas within each category based on the total revenue generated to identify the best and lowest-performing pizzas in every category.

#### SQL Concepts Used

* `WITH` Clause (CTE)
* `SUM()`
* `GROUP BY`
* `RANK()`
* `PARTITION BY`
* Window Functions
* `ORDER BY`
